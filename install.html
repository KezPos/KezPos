Step-by-Step

How to install KezPos
on your Android phone.

1

Download the APK

Tap the "Download KezPos APK" button above. Your browser will download the .apk file — check your Downloads folder once it's done.

2

Allow installation from unknown sources

Since KezPos isn't on the Play Store, Android needs your permission to install it. When prompted, tap "Allow from this source" or go to:

General path (most phones)

Settings → Apps → Special app access → Install unknown apps → Your browser → Allow

⚠️ You can turn this off again after installing for extra security.
3

Open and install the APK file

Open your Downloads folder, tap KezPos.apk, then tap "Install" on the prompt that appears. Takes about 10 seconds.

4

Open KezPos and create your account

Launch KezPos from your home screen. Enter your email and shop name to create a free account. Your data will start syncing to the cloud automatically.

✓

You're all set!

Add your first product, make your first sale, and watch your business data come alive. If you ever switch phones, sign in with the same email and everything restores automatically.

✨ Pro tip: Export your database from Settings → Backup & Data as an extra backup.

Brand-specific settings

The exact path to enable "Unknown Sources" varies by phone. Find yours below:

📱 Samsung (One UI)

  • Settings → Apps → tap top-right menu (⋮) → Special access
  • → Install unknown apps → Chrome/Files → Allow

📱 Tecno / Infinix / itel (HiOS / XOS)

  • Settings → Privacy & Safety → Install unknown apps
  • → Select your browser or Files app → Enable

📱 Xiaomi / Redmi (MIUI)

  • Settings → Additional settings → Privacy → Install unknown apps
  • → Select your browser → Allow from this source

📱 Nokia / Stock Android

  • Settings → Apps & notifications → Advanced
  • → Special app access → Install unknown apps → Allow

📱 Huawei (EMUI)

  • Settings → Security → More settings
  • → Install apps from external sources → Allow

FAQ

Common questions.

Is KezPos really free?

KezPos is priced to be accessible for local businesses. One affordable monthly plan covers everything — including cloud sync, all features, and future updates. No hidden fees, no paywalls.

Why isn't it on the Play Store?

KezPos is in active development and distributed directly via APK so updates can reach you faster. It's safe — check our Privacy Policy for details on what data we collect (spoiler: very little).

What if I lose my phone?

No problem. Sign in with the same email on your new phone and all your data — products, customers, sales history — restores automatically from the cloud.

Does it work without internet?

Yes. KezPos is offline-first. Make sales, add stock, record transactions — all without a connection. Everything syncs automatically when you reconnect.

Which Bluetooth printers are supported?

Standard 58mm Bluetooth thermal printers. Most affordable models sold in Kenya work fine. The app scans for nearby printers automatically when you connect.
